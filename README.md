# Salvadanaio D&D

## Introduzione
**Salvadanaio D&D** è un'applicazione web progettata per facilitare la gestione delle finanze per i giocatori e i Dungeon Master di *Dungeons & Dragons*. Questo strumento permette di tracciare e gestire le ricchezze dei personaggi in modo semplice ed efficiente, usando file JSON al posto di un database tradizionale.

## Caratteristiche
- Gestione semplificata dei personaggi e delle loro risorse finanziarie.
- Interfaccia utente intuitiva per operazioni rapide e dettagliate.
- Transazioni finanziarie con calcolo automatico del resto.
- Configurazione facile per personalizzare l'esperienza di gioco.
- Adatto sia a giocatori principianti che esperti di D&D.

## Inizio Rapido
1. Clona il repository sul tuo server con PHP 8.1.4 *Testato*.
2. Configura i file `settingsFE.json` e `API/config.json` secondo le tue esigenze.
3. Avvia il server e accedi all'applicazione via web browser.

## Configurazione
Modifica i file di configurazione per adattare l'applicazione:
- `settingsFE.json` per gli elementi dell'interfaccia utente.
- `API/config.json` per il tasso di cambio tra le valute.

## Uso
Una volta configurata, l'applicazione è pronta per essere usata:
- Aggiungi nuovi personaggi e gestisci le loro transazioni finanziarie.
- Visualizza cronologie dettagliate per monitorare l'evoluzione delle ricchezze dei personaggi.
